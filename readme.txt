Para probar el codigo seguir el siguiente procedimiento:

1. Crear una entorno virtual (python -m venv venv)
2. Activar el entorno virtual (venv\Scripts\activate)
3. Instalar las dependencias (pip install -r requerimientos.txt)
4. Ejecutar el script (streamlit run app.py)

NOTA: Siempre que quieras ejecutar el script debes activar el entorno virtual
